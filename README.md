# Garden App

The **Garden App** is a simple Python program designed for gardening enthusiasts around the world.  
It provides gardening advice based on the **season** and **plant type**, and offers plant recommendations for different times of the year.

This project was created to demonstrate understanding of GitHub workflows, including:
- Repository setup  
- Issue tracking  
- Branching strategy  
- Pull requests  
- Code refactoring  
- Documentation improvements
